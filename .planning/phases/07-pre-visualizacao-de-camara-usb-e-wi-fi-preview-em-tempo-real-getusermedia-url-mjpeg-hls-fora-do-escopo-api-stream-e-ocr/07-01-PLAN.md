# Plan 07-01 — Pré-visualização USB e rede (implementação)

## Objetivo
Implementar rota `/camera` com preview USB (`getUserMedia`) e preview por URL (MJPEG/`img`, HLS/`video`+`hls.js`), validação de URL e estados de UI conforme `07-UI-SPEC.md` e `07-PRD.md`.

## Tarefas
1. `src/lib/camera/validate-camera-url.ts` — validação `http(s)`, bloqueio de schemes inseguros, deteção mixed content; testes unitários.
2. `UsbCameraPreview`, `NetworkCameraPreview`, `CameraPreviewPanel` em `src/components/features/camera/`.
3. `src/app/(auth)/camera/page.tsx` — página cliente.
4. `ROUTES.AUTH.CAMERA` + item no `Sidebar` (ícone `CameraAlt` ou equivalente).
5. Dependência `hls.js`; import dinâmico no cliente para HLS.
